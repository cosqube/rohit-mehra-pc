# Jaadoo Terminal

A web-based recreation of the computer interface used to call the alien Jaadoo in the Bollywood movie *Koi Mil Gaya (2003)*.

This project is inspired by the iconic sci-fi setup shown in the film and is implemented using HTML, CSS, and JavaScript.
